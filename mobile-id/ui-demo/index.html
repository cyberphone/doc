The Web UI on the left side simulates the primary steps using an mobile "App" based authentication and signature solution having the following core characteristics:
  • Functionally similar to smart cards but without requiring readers or specfic card driver software
  • User interface in the mobile device offers a consistant user experience
  • Supports two core authentication and signature scenarios:
    • When on the mobile Web
    • Usage as a "security companion" to a PC/Mac on the Web using QR-code
  • Supports any number of independent issuers using a built-in on-line key-management system
  • Supports both PKI (X.509) and OpenID Connect login
  • State of the art security based on TEEs (Trusted Execution Environments)
