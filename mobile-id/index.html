What is Mobile ID?

Mobile ID is an effort creating a practical and secure solution for authentication and digital signatures based on mobile devices.
Unlike most other systems, Mobile ID is fully open and intended to be usable by any government, bank, enterprise, university, etc.
At a glance:
  • Supports the mobile Web and the PC/Mac Web (using the mobile device as an external authenticator)
  • The mobile application is free of charge and downloadable from an "app store"
  • Built-in support for English, French, Swedish, etc.
  • Support for any numbers of independent issuers
  • PKI and IdP mode
  • JSON based security protocol with keys stored in TEEs
Detailed technical information and source code is available at https://github.com/cyberphone/mobile-id.

Testing Mobile ID
