What is Mobile ID?

Mobile ID is an effort creating a mobile application for authentication and signatures that can be used by any government, bank, enterprise, etc.
At a glance:
  • Supports the mobile Web and the PC/Mac Web (using the mobile device as an external authenticator)
  • PKI and IdP mode
  • Built-in support for English, French, Swedish, etc.
  • JSON based security protocols
  • License free
  • Support for any numbers of independent issuers
Detailed technical information and code is available at: https://github.com/cyberphone/mobile-id

Testing Mobile ID
