Qu'est-ce que Mobile ID?

Mobile ID est un effort pour créer une application mobile pour l'authentification et des signatures pouvant être utilisées par n'importe quel gouvernement, banque, entreprise, etc.
En bref:
  • Prise en charge du Web mobile et du Web PC / Mac (en utilisant le périphérique mobile comme authentificateur externe)
  • Mode PKI et IdP
  • Prise en charge intégrée de l'anglais, du français, du suédois, etc.
  • Protocoles de sécurité basés sur JSON
  • Sans licence
  • Prise en charge d’un nombre quelconque d'émetteurs indépendants
Les informations techniques sont actuellement disponibles uniquement en anglais: https://github.com/cyberphone/mobile-id
