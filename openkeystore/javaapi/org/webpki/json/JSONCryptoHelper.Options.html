    Common JEF/JSF decoding options.

    This class holds options that are checked during decoding.

    The following options are currently recognized:
    • Algorithm preference. Default: JOSE
    • Require public key info in line. Default: true
    • keyId option. Default: FORBIDDEN
    • Permitted extensions. Default: none
    In addition, the Options class is used for defining external readers for "remoteKey" support.
