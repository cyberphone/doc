org.webpki.json

Class JSONObjectReader

  • java.lang.Object
    • org.webpki.json.JSONObjectReader
  • All Implemented Interfaces:
    Serializable, Cloneable

    public class JSONObjectReader
extends Object
implements Serializable, Cloneable
    JSON object reader.

    Returned by the parser methods. Also provides built-in support for decoding JCS (JSON Cleartext Signature) and JEF (JSON Encryption Format) constructs.

    In addition, there are methods for reading keys supplied in the JWK format.

    See Also:
    JSONParser, getObject(String), JSONArrayReader.getObject(), JSONObjectWriter.JSONObjectWriter(JSONObjectReader), Serialized Form
