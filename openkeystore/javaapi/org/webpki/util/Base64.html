      • setEncodedCharsPerRow

        public void setEncodedCharsPerRow(int count)
        Sets number of encoded characters before each linebreak. If count isn't divisible by 4, it will be treated as the nearest larger integer divisible by 4. Must be larger than zero and no more than 76. Default value is 76. Warning: Deviating from default values, might cause compatibility problems with other Base64 encoders/decoders.
        Parameters:
        count - number of base64 characters in each row

      • setLineBreakOn

        public void setLineBreakOn(boolean on)
        If set to true the base64 encoded data will be divided into a number of rows. Default value is true. Warning: Deviating from default values, might cause compatibility problems with other Base64 encoders/decoders.
        Parameters:
        on - on/off switch for line break.

      • setPaddingOn

        public void setPaddingOn(boolean on)
        If set to true the base64 encoded data will be padded at the end to make sure every 3 uncoded bytes corresponds to 4 encoded ones. Max. 2 padding bytes. Default value is true. Warning: Deviating from default values, might cause compatibility problems with other Base64 encoders/decoders.
        Parameters:
        on - on/off switch for padding.
