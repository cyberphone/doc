        public void setWebProxy​(String address,
                        int port)
                 throws IOException
        Sets HTTP web-proxy parameters. This method needs to be called for usage of the security proxy scheme where local LAN rules require outbound HTTP calls to through a web-proxy server.

        Note: The proxy scheme does currently not support web-proxy authentication.

        Parameters:
        address - The host name or IP address of the web-proxy server.
        port - The TCP port number to use.
