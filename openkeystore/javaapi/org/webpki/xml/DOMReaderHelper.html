org.webpki.xml

Class DOMReaderHelper

  • java.lang.Object
    • org.webpki.xml.DOMReaderHelper

  • public class DOMReaderHelper
extends Object
    Utility class making traversal of DOM documents easier in simple cases.

    This class holds the state of a cursor navigating the tree and provides a set of methods for accessing/converting the contents of elements.

    The cursor is moved after a successful call to getNext(), getDatatype() or getDatatypeConditional(), i.e. after the value is read.

    getChild and the DOMAttributeReaderHelper will act on the "last visited element", hereby defined to mean the element that the cursor pointed at before the last getNext(), getDatatype() or getDatatypeConditional() call, if any. This allows code like 

       helper.getNext("orderline");
   String item = helper.getAttributeHelper().getString("item");
    to extract the attribute item of element orderline.

    A small example

       DOMReaderHelper helper = new DOMReaderHelper(e);

   helper.getNext("order");

   // Go one level down in the structure.
   helper.getChild();

   buyer = helper.getString("buyer");

   buyerDUNS = helper.getAttributeHelper().getInt("duns");

   comment = helper.getStringConditional("comment");
   ...
    which parses the structure 
       <example:order xmlns:example="...">
       <buyer duns="12345">John Doe</buyer>
       <comment>Urgent!</comment>
       ...
   </example:order>
    This example code comes from an example included in the distribution to illustrate how to write an XMLObjectWrapper.

      • getAttributeHelper

        public DOMAttributeReaderHelper getAttributeHelper()
        Get the DOMAttributeReaderHelper of this DOMReaderHelper.

        The DOMAttributeReaderHelper gives access to the attributes of the "last visited element".

        Please note that the DOMAttributeReaderHelper returned will follow this DOMReaderHelper's cursor, i.e. it will always act on the current "last visited element" of this DOMReaderHelper.

        Returns:
        DOMAttributeReaderHelper
