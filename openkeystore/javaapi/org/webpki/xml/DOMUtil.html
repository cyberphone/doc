      • appendTextElement

        public static Element appendTextElement​(Element parent,
                                        String element,
                                        Object value)
        Creates a subelement with a single child, a text node, as a child of an existing node.

        The structure <element>value</element> will be created and appended to parent.

        This method is shorthand for appendTextElement(parent, element, value.toString()) and is hence useful when value is an object who's toString function returns a suitably formatted string representation, for example a StringBuilder, BigInteger or BigDecimal.

        Parameters:
        parent - The parent of the new element.
        element - The name of the new element.
        value - The value of the text node.
        Returns:
        Element
        See Also:
        appendTextElement(Element, String, String)
