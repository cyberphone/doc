org.webpki.xml

Class DOMAttributeReaderHelper

  • java.lang.Object
    • org.webpki.xml.DOMAttributeReaderHelper

  • public class DOMAttributeReaderHelper
extends Object
    Utility class making traversal of DOM documents easier in simple cases.

    The general use of this class is described in the documentation for DOMReaderHelper.

    Please note that the DOMAttributeReaderHelper returned by DOMReaderHelper.getAttributeHelper() will follow the DOMReaderHelper's cursor, i.e. it will always act on the current "last visited element" of the DOMReaderHelper.

    See Also:
    DOMReaderHelper, DOMReaderHelper.getAttributeHelper()
