Class XMLObjectWrapper

  • java.lang.Object
    • org.webpki.xml.XMLObjectWrapper
  • Direct Known Subclasses:
    XMLSignatureWrapper
    public abstract class XMLObjectWrapper
extends Object
    Base class for java classes whos data can be translated to/from XML documents.

    A XMLObjectWrapper subclass represents (and handles) a certain schema.

    A XMLObjectWrapper subclass should be capable of initializing from a DOM Document and also produce a DOM Document representing it's current state. It must have a public empty (no argument-) constructor to allow the XMLSchemaCache to create an object instance and call it's namespace and element methods.

    A short example of an XMLObjectWrapper using DOMReaderHelper and DOMWriterHelper to handle it's XML representation is supplied with the distribution.
