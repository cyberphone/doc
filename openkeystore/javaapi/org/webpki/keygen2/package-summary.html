Package org.webpki.keygen2 Description

KeyGen2 - Credential Provisioning and Management Protocol

KeyGen2 is an JSON-based, Internet-browser-hosted protocol for provisioning and managing resources in a "companion specification" coined SKS (Secure Key Store).
