Personal Payment Terminal
The following UI simulates the final operation of the Android version of the Personal Payment Terminal.
  • Eliminates the traditional payment terminal
  • Supports most payee-initiated payment scenarios including:
    • Local payments in a shop
    • Payments using the mobile Web
    • Usage as a payment "companion" to a PC/Mac on the Web
  • Maintains a security-level comparable to EMV-cards
